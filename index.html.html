How to Play:

  • • Drag each asset from the left panel to its correct position on the Risk-Return matrix
  • • X-axis represents Return (left = low, right = high)
  • • Y-axis represents Risk (bottom = low, top = high)
  • • Correct placements stay organized in their quadrant, incorrect placements return to the pool
  • • Get +10 points for each correct placement, -2 points for incorrect placements
